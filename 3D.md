# Volume Explorer — 重建邏輯參考

在你自己的 Next.js 專案中重新實作這個「3D 體積學習 + AI Tutor」功能。
**不需要照原本的檔案結構**，自由組織即可。下面只給整體邏輯與必要的 import。

---

## 1. 套件

```powershell
npm install three @react-three/fiber @react-three/drei zustand immer ai @openrouter/sdk zod
npm install -D @types/three
```

`next.config.ts` 加入（**必須**，否則 three.js 會壞）：
```ts
transpilePackages: ['three', '@react-three/fiber', '@react-three/drei'],
```

---

## 2. 整體架構

```
┌─────────────────────────────────────────────────┐
│  Page (client component)                        │
│  ├── 3D Canvas    ──► 顯示 / 操作立方體          │
│  ├── 控制面板      ──► 切換模式、語言、工具      │
│  ├── 體積顯示      ──► 即時計算                  │
│  └── AI Tutor 對話 ──► 出題 / 批改 / 提示        │
└──────────────┬──────────────────────────────────┘
               │
        ┌──────┴──────┐
        ▼             ▼
   Zustand Store    API Route
   (前端狀態)        (server)
                       │
                       ▼
                  AI SDK + OpenRouter
                  + tools (Zod schema)
```

## 3. 三大邏輯模組

### A. 幾何與狀態（純邏輯，不依賴 React）

**職責：** 用 `(x,y,z)` 座標管理立方體集合；提供新增/刪除/計算體積/三視圖投影。

```ts
import { create } from "zustand";
import { immer } from "zustand/middleware/immer";

type Cube = { x: number; y: number; z: number };

// 體積 = 立方體數量；表面積、三視圖另寫純函式
function computeVolume(cubes: Cube[]): number { /* ... */ }
function projectViews(cubes: Cube[]) { /* top / front / side */ }

export const useCubeStore = create(immer<{
  cubes: Cube[];
  add: (c: Cube) => void;
  remove: (c: Cube) => void;
  clear: () => void;
}>(/* ... */));
```

全域 UI 狀態（語言、模式）另開一個 store：
```ts
import { create } from "zustand";
export const useAppStore = create<{
  language: "zh" | "en";
  mode: "build" | "inspect";
  // setters...
}>(/* ... */);
```

---

### B. 3D 場景（React Three Fiber）

**職責：** 把 store 裡的立方體陣列渲染成 3D；點擊 raycast 後加/減立方體。

```tsx
"use client";
import { Canvas } from "@react-three/fiber";
import { OrbitControls, Grid } from "@react-three/drei";
import * as THREE from "three";
import { useCubeStore } from "@/store/cubeStore";

function Scene() {
  const cubes = useCubeStore((s) => s.cubes);
  return (
    <Canvas camera={{ position: [5, 5, 5] }}>
      <ambientLight />
      <directionalLight position={[5, 10, 5]} />
      <Grid args={[10, 10]} />
      {cubes.map((c) => (
        <mesh key={`${c.x},${c.y},${c.z}`} position={[c.x, c.y, c.z]}>
          <boxGeometry />
          <meshStandardMaterial />
        </mesh>
      ))}
      <OrbitControls />
    </Canvas>
  );
}
```

R3F 的 JSX type 宣告（任意 `.d.ts`，例如 `app/r3f.d.ts`）：
```ts
import { ThreeElements } from "@react-three/fiber";
declare global {
  namespace JSX { interface IntrinsicElements extends ThreeElements {} }
}
```

> R3F 元件**只能在 client component** 使用。Page 加 `"use client"`，或用 `next/dynamic` + `{ ssr: false }`。

### C. AI Tutor（Vercel AI SDK + OpenRouter + tools）

**職責：** LLM 出題 / 批改 / 給提示。用 tool calling 讓模型結構化輸出。

**Server side（API route）：**
```ts
// app/api/tutor/route.ts
import { NextRequest } from "next/server";
import { streamText, tool } from "ai";
import { createOpenRouter } from "@openrouter/sdk";
import { z } from "zod";

const openrouter = createOpenRouter({ apiKey: process.env.OPENROUTER_API_KEY! });

const tools = {
  askQuestion: tool({
    description: "出一道體積題目",
    parameters: z.object({ difficulty: z.enum(["easy", "medium", "hard"]) }),
    execute: async ({ difficulty }) => ({ question: "...", answer: 12 }),
  }),
  checkAnswer: tool({
    description: "批改學生答案",
    parameters: z.object({ userAnswer: z.number(), correctAnswer: z.number() }),
    execute: async (a) => ({ correct: a.userAnswer === a.correctAnswer }),
  }),
  giveHint: tool({
    description: "給提示",
    parameters: z.object({ topic: z.string() }),
    execute: async ({ topic }) => ({ hint: "..." }),
  }),
};

export async function POST(req: NextRequest) {
  const { messages } = await req.json();
  const result = streamText({
    model: openrouter("openai/gpt-4o-mini"),
    system: "你是國小體積家教...",
    messages,
    tools,
    maxSteps: 5,
  });
  return result.toDataStreamResponse();
}
```

**Client side（聊天面板）：**
```tsx
"use client";
import { useChat } from "ai/react";

function TutorPanel() {
  const { messages, input, handleInputChange, handleSubmit } = useChat({
    api: "/api/tutor",
  });
  // 渲染訊息列表 + 輸入框
}
```

## 4. 必裝 import 清單（記住這些就夠）

| 用途 | import |

| AI Server | `import { streamText, tool } from "ai"` <br> `import { createOpenRouter } from "@openrouter/sdk"` <br> `import { z } from "zod"` |

```dotenv
OPENROUTER_API_KEY=sk-or-v1-xxxxxxxxxxxx
```

## 6. 開發順序

1. **純函式**：`computeVolume`、`projectViews` → 寫單元測試
2. **Store**：`useCubeStore`、`useAppStore`
3. **3D 場景**：渲染 + 點擊加減立方體
4. **UI 殼**：模式切換、語言切換、體積顯示
5. **AI API route** + tools
6. **TutorPanel** 串接前後端

第 1–4 步完成就有可玩的 3D 體積 App；5–6 才需要 OpenRouter key。
